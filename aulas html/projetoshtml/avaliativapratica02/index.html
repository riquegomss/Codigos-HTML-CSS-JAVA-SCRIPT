JAVA SCRIPT

JavaScript é uma linguagem de programação de alto nível, criada em 1995 por Brendan Eich, um funcionário da Netscape, empresa que criou um dos primeiros navegadores da web. É uma linguagem interpretada e baseada em eventos, amplamente usada para desenvolvimento web. Inicialmente criada para tornar páginas da web interativas e para fazer validações no lado cliente (front-end). Atualmente é usada em diversas áreas, incluindo servidores, aplicativos móveis, jogos e até inteligência artificial.

CARACTERÍSTICAS

São caracaterística dessa linguagem: multiparadigma, pois suporta programação funcional, orientada a objetos e baseada em eventos; interpretada, o código JavaScript é executado diretamente pelo navegador (o runtime), ou por um ambiente como o Node.js, sem necessidade de compilação. dinamicamente tipada, as variáveis podem armazenar diferentes tipos de dados sem necessidade de declaração explícita e juntamente com HTML e CSS, é uma das principais tecnologias da web.

VERSÕES

O JavaScript está em evolução, sua última versão é o ECMAScript 13 (ES13) de 2022, nessa versão aconteceram melhorias no acesso a índices de arrays e alterações relaionadas ao Node.js. A versão ECMAScript 3 de 1999, trouxe suporte a tratamento de erros (try/catch) e manipulação de strings, a versão de ECMAScript 5 (ES5) de 2009 deu suporte para JSON nativo, introdução dos métodos como forEach(), map(), filter(), reduce() e melhorias de segurança e desempenho. Houve uma grande atualização em 2015, o ECMAScript 6, nessa versão foram introduzidos o comando let e const para declarar variáveis, classes e módulos (import/export) para organização do código, entre outras. Muitas versões vieram depois, como:ECMAScript 7 em 2016, ECMAScript 8 em 2017, ECMAScript 9 em 2018, ECMAScript 10 em 2019, ECMAScript 11 em 2020 e ECMAScript 12 em 2021.
